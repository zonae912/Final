# Campus Resource Hub - Wireframes

## Overview
This document contains wireframes for the main screens of Campus Resource Hub. These are ASCII-based wireframes showing the layout and component structure. For submission, these can be hand-sketched and scanned or converted to proper mockups using tools like Figma, Balsamiq, or draw.io.

## Design Specifications

### Color Palette
- Primary: #0056b3 (Blue - trust, education)
- Secondary: #28a745 (Green - success, available)
- Accent: #ffc107 (Amber - warnings, pending)
- Danger: #dc3545 (Red - errors, conflicts)
- Neutral: #6c757d (Gray - text, borders)
- Background: #f8f9fa (Light gray)
- Text: #212529 (Dark gray)

### Typography
- Headings: System font stack (Segoe UI, Roboto, Helvetica Neue)
- Body: 16px, line-height 1.5
- Buttons: 14px, font-weight 500
- Labels: 14px, font-weight 600

### Spacing
- Base unit: 8px
- Component padding: 16px
- Section margins: 24px
- Card spacing: 16px

### Responsive Breakpoints
- Mobile: < 768px (single column, hamburger menu)
- Tablet: 768px - 1024px (two columns, collapsible sidebar)
- Desktop: > 1024px (full layout as shown)

### Accessibility
- WCAG 2.1 Level AA compliance
- Color contrast ratio 4.5:1 minimum
- Focus indicators on all interactive elements
- Screen reader labels on icons
- Keyboard navigation support
- Skip to main content link

## Implementation Notes

1. **Icons**: Use Bootstrap Icons or Font Awesome for consistency
2. **Images**: Placeholder service (e.g., via.placeholder.com) for development
3. **Forms**: HTML5 validation + client-side JavaScript validation
4. **Modals**: Bootstrap modal component for booking/message forms
5. **Calendar**: FullCalendar.js or custom grid implementation
6. **Image Carousel**: Bootstrap carousel or Swiper.js
7. **Filters**: JavaScript for instant filtering, AJAX for server-side updates
8. **Pagination**: Server-side pagination with page size of 12-24 items
